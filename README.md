### Description of Website:
**Rotisserie Rumble** is a website produced with Flask and SQLite3 that allows users to sign up for competitive card games.
Users can spend their money and compete against the House in various card games accessed through our catalog. Users can increase their monetary balance and change their profile.

### Install Guide:
1. **Clone the repository**: `git clone git@github.com:jliu444/RotisserieChicken.git`
2. **Navigate into directory**: `cd RotisserieChicken`
3. **Create virtual environment**: `python3 -m venv venv`
4. **Activate virtual environment**:
  - maxOs/Linux: `. venv/bin/activate`
  - Windows: `venv\Scripts\activate`
5. **Install dependencies**: `pip install -r requirements.txt`

### Launch Codes:
In the terminal, access the project root directory (app/) and run the following commands:
- `~$ python build_db.py`
- `~$ python __init__.py`

### Launch Website
Using the provided, locally hosted link, access the website. To deactivate the virtual environment, type `deactivate`.

### Gaining Access
- To gain access to the card game collection, you must sign up / log into your account
- Once authenticated, you'll be brought to the profile page, where you can change your profile and add to your money balance
- In the catalog page you'll be able to select your card game of choice
- In the game you'll be able to bet money and win / lose
